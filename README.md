# Nir Cheti System

Personal AI system for managing:
- Daily logs
- Ideas
- Projects
- System memory
- Work protocols

## Structure

- /daily → daily logs
- /ideas → ideas parking
- /projects → active projects
- /system → stable memory
- /templates → templates

## Purpose

Create a structured thinking system that connects:
Daily → Ideas → Projects → Execution

## Information Flow

- Daily Log → temporary work layer
- Ideas Parking Lot → keeps valuable ideas
- Project Index → tracks active initiatives
- System Memory Stable → stores fixed rules and long-term truths
- Master File → end-of-day summary and permanent daily record

### Rule
Not everything stays in Daily.
Daily is for work.
Only valuable outputs move forward.
